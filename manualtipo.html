La Segunda, plantas y flores

El objetivo de este proyecto es generar la identidad de marca de una plantería o florería. El concepto se tiene que reflejar a lo largo de las aplicaciones. El uso de elementos gráficos como la fotografía o las formas son esenciales para el desarrollo del mismo. Se empieza por la investigación del tema, el cual por instrucciones del proyecto, debe estar directamente ligado a la familia de quien lleva a cabo el proyecto. Después se lleva a cabo un proceso de bocetaje para crear el logotipo y más adelante los diseños de las aplicaciones como vasos de café, delantales y/ó comunicación externa.
